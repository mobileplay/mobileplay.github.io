How to find your TV model code
  • Samsung: Settings → Support → About This TV → Model Code
  • LG: Settings → General → About This TV → Model Name
  • Sony: Settings → System → About → Model
  • TCL: Settings → System → About → Model
  • Hisense: Settings → Support / System → About → Model
  • Roku / Fire TV: Settings → System → About → Model / Device info
  • Other: Check Settings → About / Support on your TV
