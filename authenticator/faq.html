What is the Authenticator app?

The Authenticator app generates secure two-factor authentication (2FA) codes for your accounts. When you set up your account with 2FA, you will receive a secret key to enter into the Authenticator app, usually in the form of a QR code. This establishes a secure connection between the Authenticator app and your account. Once the connection is established, the app will generate a 6-8 digit code that is required to access your account. Even if someone knows your password, they still need the 2FA code to access your account.

What is Two-Factor Authentication (2FA)?

Two-Factor Authentication (2FA) is an extra layer of security for your account in which users provide two different authentication factors to verify themselves. This process is done to better protect both the user's credentials and the resources the user can access. 2FA provides a higher level of security than single-factor authentication (SFA), where the user provides only one factor, typically a password or passcode. 2FA methods rely on the user providing a password, as well as a second factor, usually a security code or a biometric factor such as a fingerprint or facial scan.
