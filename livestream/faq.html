HOW TO USE?

This page will help you understand how to use this app

  1. SCREEN RECORDER
    • Open Stream & Cast App
    • Screen Recording
    • Start Recording
    • Now you can leave the app and start recording the screen
  2. LIVE STREAM
    • Open Stream & Cast App
    • Stream Now
    • Select Camera Source
    • Input Server URL Stream Key
    • Start Streaming
  3. LIVE GAME
    • Open Stream & Cast App
    • Stream Now
    • Select Screen Source
    • Input Server URL Stream Key
    • Start Streaming
